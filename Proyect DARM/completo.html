Lista de definiciones:

HTML
Define cómo se organizan los textos, imágenes, videos y otros elementos de una página web.
CSS
Permite definir el aspecto de los elementos de una página, como el color, la fuente, el tamaño, el fondo, los márgenes y la disposición de los elementos.
JavaScript
Es un lenguaje de programación que permite crear interactividad en una página web.
